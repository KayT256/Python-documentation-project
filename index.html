Introduction

Python is a high-level, interpreted programming language known for its simplicity and versatility. It was created by Guido van Rossum and first released in 1991. Python's design philosophy emphasizes code readability through the use of significant whitespace. This makes it an excellent language for beginners and experts alike.

Python is used in a wide range of applications, from web development to data analysis, machine learning, artificial intelligence, scientific computing, and more. One of its key benefits is the vast standard library and the extensive ecosystem of third-party packages, which provide tools for nearly every task. Python's simplicity allows beginners to quickly start writing programs, while its depth supports complex applications for advanced users.

Basic concepts
The "Hello, World!" Program

One of the simplest Python programs you can write is the "Hello, World!" program. It demonstrates how to output text to the console

print("Hello, World!");

# Output: Hello, World!
Syntax Basics
  • Indentation: Python uses indentation to define blocks of code. Consistent spacing is crucial, as different indentations signal different blocks. For example, all code within a loop or function must be indented under that loop or function:
    if True:
        print("This is indented, so it's part of the if statement.")

    # Output: This is indented, so it's part of the if statement.
  • Variables: Variables are used to store information that can be referenced and manipulated in your program. In Python, you declare a variable by assigning it a value with the = operator:
    my_variable = 10
  • Data Types: Python has several data types, including integers int, floating-point numbers float, strings str, and booleans bool. Python is dynamically typed, so you don't need to declare the type of a variable explicitly.
    an_integer = 5
    a_float = 5.0
    a_string = "Hello"
    a_boolean = True
  • Operators: Python includes operators for calculations and comparisons. For example, arithmetic operators include +, -, *, / for addition, subtraction, multiplication, and division. Comparison operators include ==, !=, >, <, >=, <= for equality, inequality, greater than, less than, greater than or equal to, and less than or equal to, respectively.
    # Arithmetic operation
    result = 5 + 3 # Equals 8

    # Comparison
    is_equal = (result == 8) # Evaluates to True
Control structures
Conditional Statements and Loops
  • Conditional Statements: Python uses if, elif (else if), and else to execute code based on conditions. These statements allow you to branch your code depending on whether conditions are true or false.
    age = 18
    if age >= 18:
        print("You are an adult.")
    elif age > 13:
        print("You are a teenager.")
    else:
        print("You are a child.")

    # Output: You are an adult.
  • Loops: Python provides for and while loops to iterate over a sequence (like a list, tuple, or string) or execute a block of code multiple times.
    • For Loop: Used for iterating over a sequence
      fruits = ["apple", "banana", "cherry"]
      for fruit in fruits:
          print(fruit)

      # Output:
      #      apple
      #      banana
      #      cherry
    • While Loop: Repeats a block of code as long as a condition is true
      count = 0
      while count < 5:
          print(count)
          count += 1 # This is the same as count = count + 1

      # Output:
      #      0
      #      1
      #      2
      #      3
      #      4
Using break, continue, and pass
  • break: Exits the loop
  • continue: Skips the rest of the code inside the loop for the current iteration and moves to the next iteration
  • pass: Does nothing; used as a placeholder in situations where syntax requires a statement but you do not want any command or code to execute
    • # In this example, numbers will be printed until the loop reaches 5.
    # Even numbers are skipped due to the continue statement.
    for number in range(1, 10):
        if number == 5:
            break # Exit the loop when number is 5
        if number % 2 == 0:
            continue # Skip even numbers

    # Output:
    #      2
    #      4
Functions
Defining Functions, Arguments, and Return Values

Functions in Python are defined using the def keyword, followed by the function name and parentheses () containing any parameters. Functions allow you to encapsulate a task, making it reusable and organized.

def greet(name):
    return f"Hello, {name}!"

To call a function, use its name followed by parentheses. If the function expects arguments, provide them inside the parentheses:

message = greet("Alice")
print(message)

# Output: Hello, Alice!
Scope of Variables

Variables in Python have a scope that determines where they can be accessed. The two main scopes are:

  • Local Scope: Variables created inside a function belong to the local scope of that function, and can only be used inside that function.
  • Global Scope: Variables created in the main body of a Python script are global and can be accessed anywhere in the script, including by functions.
    • x = "global" # Global scope

    def myfunc():
        y = "local" # Local scope
        print(x) # Accesses the global variable
        print(y) # Accesses the local variable

    myfunc()
    print(x) # Works
    # print(y) # This would raise an error because y is not accessible here

    # Output:
    #     global
    #     local
    #     global
Data Structures
Overview of Lists, Tuples, Dictionaries, and Sets
  • Lists: Ordered collections of items that can be changed (mutable). Lists are created by placing items inside square brackets [], separated by commas.
    fruits = ["apple", "banana", "cherry"]
    fruits.append("orange") # Adding an item
    print(fruits)

    # Output: ["apple", "banana", "cherry", "orange"]
  • Tuples: Ordered collections of items that cannot be changed (immutable). Tuples are created by placing items inside parentheses (), separated by commas.
    coordinates = (10.0, 20.0)
    print(coordinates)

    # Output: (10.0, 20.0)
  • Dictionaries: Unordered collections of key-value pairs. Dictionaries are created by placing items inside curly braces {}, with each key separated from its value by a colon :.
    person = {"name": "Alice", "age": 30}
    person["email"] = "alice@example.com" # Adding a new key-value pair
    print(person)

    # Output: {"name": "Alice", "age": 30, "email": "alice@example.com"}
  • Sets: Unordered collections of unique items. Sets are created by placing items inside curly braces {}, separated by commas. They are similar to dictionaries but only contain keys.
    colors = {"red", "green", "blue"}
    colors.add("yellow") # Adding an item
    print(colors)

    # Output: {"red", "green", "blue", "yellow"}
Basic Operations: Indexing, Slicing, and Iterating
  • Indexing: Accessing an item in a list or tuple by its position.
    print(fruits[1])

    # Output: banana
  • Slicing: Accessing a subset of a list or tuple.
    print(fruits[1:3])

    # Output: ['banana', 'cherry']
  • Iterating: Looping through each item in a list, tuple, dictionary, or set.
    for fruit in fruits:
        print(fruit)

    for key, value in person.items():
         print(f"{key}: {value}")

    # Output:
    #     ["apple", "banana", "cherry", "orange"]
    #     name: Alice
    #     age: 30
    #     email: alice@example.com
